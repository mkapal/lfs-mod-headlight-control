LFS AI mod headlight control
=======================

This application automatically turns on headlights for local AI drivers every time a session starts or restarts (race, practice or qualifying).

Configuration
=============

1. Copy config.toml to config.local.toml - this will be your custom configuration.
2. Open config.local.toml.
3. Fill in your LFS host, InSim port and admin password. You must add the IP address where this application will be running at to the Allowed IPs in the host access control panel at lfs.net.
4. Edit the list of AI names to your liking. Those names will be detected by this application to activate their headlights.
